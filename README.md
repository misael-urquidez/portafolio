# 🎮 Portafolio — Misael Urquidez Arredondo

> Portafolio personal con estética pixel art, ciclo día/noche dinámico y animaciones de scroll. Construido con HTML, CSS y JavaScript puro — sin frameworks, sin dependencias.

---

## ✨ Features

- **Ciclo día → noche** generado en tiempo real con `requestAnimationFrame` conforme el usuario hace scroll — el cielo, sol, luna y estrellas transicionan de forma continua y suave
- **Avatar animado** con efecto de bob pixelado y glow al hacer hover
- **Typing effect** con frases rotativas en el hero
- **Scroll reveal** con animaciones escalonadas y curva `cubic-bezier` profesional — cada sección entra desde una dirección distinta
- **Cursor personalizado** con partículas al hacer clic
- **Barra de progreso** de lectura fija en la parte superior
- **Nubes flotantes** y estrellas generadas dinámicamente
- Responsive para móvil y desktop

## 🚀 Cómo usar

No requiere instalación ni servidor. Solo abre el archivo directamente en el navegador:

```bash
# Clona el repo
git clone https://github.com/MisaelUrquidez/portafolio.git

# Entra a la carpeta
cd portafolio

# Abre en el navegador (o con Live Server en VS Code)
open portafolio.html
```
